# HGCGMA

This repository is the implementation of "*Optimal Model Averaging for Heterogeneous Gaussian Copula Graphical Models*".

**File Structure:**

- **`main_functions`**
  - HGCGMA.r: main function of the proposed algorithm;
  - data_generate.r: function for simulation data generation;
  - estimate_Omega.r: 
  - cov_est.r: auxiliary function;
  - Omega_est.r: auxiliary function;
    
- **`sim_codes`**  
  - Figure_1.r: code for generating Figure 1;
  - Figure_2_and_Table_3.r: code for generating Figure 2 and Table 3;
  - Table_2.r: code for generating Table 2;
  - Figure_S1.r: code for generating Figure S1;
  - Table_S1.r: code for generating Table S1.
